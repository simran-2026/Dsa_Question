# Daily DSA Questions

Welcome to my Daily DSA Questions repository! Here, I post one Data Structures and Algorithms question every day to help improve problem-solving skills.

## Topics Covered

### Data Structures
- Arrays
- Strings
- Linked Lists
- Stacks
- Queues
- Trees
- Binary Trees
- Binary Search Trees
- Heaps/Priority Queues
- Hash Tables
- Graphs
- Tries
- Segment Trees
- Disjoint Set Union (DSU)

### Algorithms
- Sorting Algorithms
- Searching Algorithms
- Recursion
- Backtracking
- Dynamic Programming
- Greedy Algorithms
- Divide and Conquer
- Bit Manipulation
- Sliding Window
- Two Pointers
- Fast & Slow Pointers
- Merge Intervals
- Cyclic Sort
- Topological Sort
- Graph Algorithms (BFS, DFS, Dijkstra, etc.)

## How to Use
1. Check the daily question in the repository
2. Try to solve it on your own
3. Compare with the solution (if provided)
4. Repeat daily!
